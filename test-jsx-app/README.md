# test-jsx-app

A TUIX application created with the jsx template.

## Getting Started

```bash
# Install dependencies
bun install

# Run the app
bun run dev

# Or run directly
./app.tsx
```

## Features

- ⚛️  JSX-based UI components
- 🎨 Rich terminal styling and layouts
- ⌨️  Keyboard and mouse input handling
- 🔧 TypeScript support
- ⚡ Fast development with Bun
